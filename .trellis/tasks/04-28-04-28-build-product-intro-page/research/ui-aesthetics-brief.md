# UI Aesthetics Brief

## Skill Route

- Route: Generation

## Design Priorities

- Improve hierarchy, spacing, and typography before decoration.
- Use accents, glow, and blur to clarify emphasis, not to cover weak layout.
- Keep motion controlled and product-like instead of theatrical.
- Preserve a crisp, shippable appearance.

## Applied Direction For This Task

- Build a single-file product intro page in Chinese.
- Use a luminous warm palette instead of a flat monochrome style.
- Favor a premium launch-page feel: radiant gradients, glass panels, layered card depth, subtle hover lift, and spotlight backgrounds.
- Keep CTA visibility high and sections easy to scan.

## Avoid

- Generic centered white-page SaaS composition
- Overdense copy blocks
- Random color usage without a coherent system
- Constant motion on every element
